## Overview

There are three folders in this github.

1. `uknit-constructions` : contains the code for the uKNIT framework

2. `UKNITBC_ANALYSIS` : contains the code for the security analysis of uKNIT-BC

3. `uknit-implementations` : contains the source code for the implementation of uKNIT-BC in `.cpp` and `.sv` as well as the test vectors
