# uKNITBC implementations
This repository contains the implementations and test vectors for `uKNIT-BC` block cipher
Currently, there are four files:
1. `uKNIT-BC.cpp` -- contains the uKNIT-BC implementation and the file for generation of test vectors.
2. `uKNIT-BC.sv`  -- contains the uKNIT-BC implementation in hardware description language.
3. `test-vectors.txt` -- contains the test vectors of uKNIT-BC.
4. `uKNIT-BC_alternative_representation.cpp` -- contains the uKNIT-BC implementation using an alternative representation describe in Appendix C of the paper
